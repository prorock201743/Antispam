# 🤖 AntiFlood Bot для Telegram

Бот для защиты групп от флуда с гибкими настройками через инлайн-меню.

## ✨ Возможности

- **Подсчёт сообщений** за настраиваемый интервал (1 час — 7 дней)
- **3 режима ограничения**: мут, кик, бан
- **Белый список** — пользователи, которых бот игнорирует
- **Автоудаление** предупреждений бота через заданное время
- **Управление через инлайн-кнопки** прямо в Telegram
- **Многогрупповое управление** — один бот, несколько групп
- **Статистика нарушителей**

## 🚀 Установка

### 1. Создание бота

1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot`
2. Получите токен и вставьте в `.env`

### 2. Настройка

```bash
# Клонировать / распаковать проект
cd antiflood_bot

# Создать .env
cp .env.example .env
nano .env  # Вставьте BOT_TOKEN и BOT_ADMINS
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск

```bash
python main.py
```

## ⚙️ Настройка по умолчанию

| Параметр              | Значение       |
|-----------------------|----------------|
| Интервал              | 1 сутки        |
| Лимит сообщений       | 3              |
| Режим ограничения     | Мут            |
| Длительность мута     | 1 сутки        |
| Автоудаление сообщ.   | Вкл (90 сек.)  |

## 📋 Команды

| Команда           | Где          | Описание                          |
|-------------------|--------------|-----------------------------------|
| `/start`          | ЛС           | Приветствие                       |
| `/groups`         | ЛС           | Список групп и настройки          |
| `/help`           | ЛС / группа  | Справка                           |
| `/antiflood`      | Группа       | Открыть настройки (для адм.)      |
| `/reset @user`    | Группа       | Сбросить нарушения пользователя   |

## 🏗 Структура проекта

```
antiflood_bot/
├── main.py                        # Точка входа
├── config.py                      # Конфигурация
├── requirements.txt

    ├── handlers/

    ├── keyboards/
    │   └── keyboards.py           # Инлайн-клавиатуры
    └── database/
        └── db.py                  # SQLite база данных
```

## 🔧 Добавление бота в группу

1. Добавьте бота в группу
2. Назначьте его **администратором** с правами:
   - Удалять сообщения
   - Ограничивать пользователей
3. Бот автоматически зарегистрирует группу
